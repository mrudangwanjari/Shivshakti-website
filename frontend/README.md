# Shivshakti Gurukulam - Frontend

This is the frontend for the Shivshakti Gurukulam website.

## Structure

- `index.html` - Main HTML file with embedded CSS and JavaScript

## Features

- Responsive design with mobile menu
- Multi-language support (English, Hindi, Marathi)
- Scroll animations
- Training programs showcase
- Women empowerment section
- Registration form
- Gallery section
- Contact information

## To Run

Simply open `index.html` in a web browser.

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Cinzel, Philosopher, Noto Sans Devanagari)

## Sections

1. Navigation Bar
2. Hero Section
3. Stats Strip
4. About Section
5. Training Programs
6. Women Empowerment
7. Gallery
8. Join/Registration Form
9. Contact Section
10. Footer
11. Language Switcher

## Notes

- The design uses a traditional Indian color scheme with gold, saffron, and green
- All content is ready for multi-language translation
- Form submissions need backend integration
- Image placeholders are included for gallery and logo
